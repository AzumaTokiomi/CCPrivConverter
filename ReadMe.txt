■概要
	『CCぷらいべった～こんば～た～』は、ココフォリアで出力できるHTMLログを、
	Privatter（Privatter+）向けに変換するツールです。

■注意
	このソフトウェアの使用により発生した問題について、作者は責任を負いません。
	自己責任で使用お願いいたします。

■動作環境
	Windows11で動作確認済み。
	
	※MacはPyInstallerに対応していないためexe形式の実行はできません。
	　ただし、pythonの実行環境が整っていれば、ソースコードから起動できる可能性があります。
	　詳しい人はGitHubからコードを取得し、pythonから実行してください。

■内容物
	・ReadMe.txt…このテキストファイルです。
	・CCPrivConverter.exe…実行ファイルです。
	　・output…実行時に生成されるフォルダです。変換後ログやエラーログが保存されます。
	　・settings…実行時に生成されるフォルダです。設定ファイルが保存されます。

■インストール
	インストールなんてものはなく、ただexeファイルを実行するだけなのですが、
	WindowsDefenderやウィルス対策ソフトに警告されることがあります。

	怪しい動作は一切していませんが、警告が出た場合は「詳細情報」→「実行」で起動してください。
	ただし、安全性については最終的にご自身の判断でご利用ください。

■使い方
	ココフォリアのログ（.html）のパスを指定した後に「ログ変換」ボタンを押すだけです。
	画像付きの細かな機能解説はNotion上のマニュアルをご確認ください。

	マニュアルURL：
	https://www.notion.so/CC-1febe1a0ab1380f2a47ff942aa33334e?pvs=4
